Install
=======
`winget install GoLang.Go` \
`winget install Git.Git` \
`go install github.com/cheat/cheat/cmd/cheat@latest` \
`cd $env:APPDATA; git clone https://github.com/p-jach/cheat.git`

Usage
-----
`cheat -l` \
`cheat net` \
`cheat -s net`
