Installing
==========

Install GoLang
--------------
`winget install GoLang.Go`

Install git
-----------
`winget search Git.Git`

Install cheat package
---------------------
`go install github.com/cheat/cheat/cmd/cheat@latest`

Pull cheatsheets
----------------
`git clone https://github.com/p-jach/cheat.git`
